New: read / write SpacingInterface.proportional_notation_duration attribute.

You can now set the public proportional_notation_duration attribute
on any Abjad score for scorewide proportional notation duration.

   t = Score([Staff(construct.scale(4))])
   t.spacing.strict_grace_spacing = True
   t.spacing.strict_note_spacing = True
   t.spacing.uniform_stretching = True
   t.spacing.proportional_notation_duration = Rational(1, 56)

   \new Score \with {
           \override SpacingSpanner #'strict-grace-spacing = ##t
           \override SpacingSpanner #'strict-note-spacing = ##t
           \override SpacingSpanner #'uniform-stretching = ##t
           \set proportionalNotationDuration = #(ly:make-moment 1 56)
   } <<
           \new Staff {
                   c'8
                   d'8
                   e'8
                   f'8
           }
   >>
